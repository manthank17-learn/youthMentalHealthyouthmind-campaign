# YouthMind – Youth Mental Health Awareness Website  
Coursework Submission for Internet Technology & Web Design

This project is a basic multi-page website created to fulfill the coursework requirement of designing a simple website using HTML, CSS, and basic JavaScript.  
The website is themed around a youth mental health campaign called **YouthMind**.

---

## Pages Included

1. **Home (index.html)**  
   Introduction to the campaign, mission statement, featured image, and main call to action.

2. **About Us (about.html)**  
   Background of the initiative, team members, and goals of the campaign.

3. **Resources (resources.html)**  
   List of available mental health support options including peer support, helplines, workshops, and a downloads section.

4. **Gallery (gallery.html)**  
   Collection of images representing events, outreach activities, and testimonials.

5. **Contact Us (contact.html)**  
   A simple contact form layout for inquiries, volunteer sign-ups, or feedback  
   (form is for demonstration only; it does not submit to a backend).

---

## Technologies Used

- **HTML5** – Structure of the pages  
- **CSS3** – Styling and layout  
- **JavaScript (very basic)** – Mobile navigation toggle and footer year  
- **GitHub Pages** – Used to host and display the website online
